# 🎵 Party Mode Audio Setup

## 📁 Where to place your music file:

```
public/assets/audio/party-music.mp3
```

## 🎶 Music Requirements:

- **Format**: MP3 (recommended)
- **Duration**: 15-30 seconds (will loop)
- **Size**: Under 2MB for fast loading
- **Style**: Upbeat, party, electronic, or fun music

## 🎵 Suggested Music Types:

- Electronic dance music
- Upbeat pop songs
- Party remixes
- Tropical house
- Fun instrumental tracks

## 📝 How to Add Music:

1. **Place your file**:
   ```
   Copy your music file → public/assets/audio/party-music.mp3
   ```

2. **File must be named exactly**: `party-music.mp3`

3. **Refresh the app** and test Party Mode!

## 🎮 How Party Mode Audio Works:

- When you click "Party Mode" → Music starts automatically
- Music loops continuously for 5 seconds
- Audio control appears at bottom-right
- Click ❌ to stop music early
- Volume is set to 50% by default

## 🔧 Troubleshooting:

**Music not playing?**
- Check file is at correct path: `/assets/audio/party-music.mp3`
- Ensure browser allows audio (some browsers require user interaction first)
- Check console for audio errors

**File too large?**
- Compress audio to under 2MB
- Use shorter loop (15-30 seconds)

## 🎨 Pro Tips:

- Choose music with good energy
- Loop seamlessly if possible
- Test in different browsers
- Consider copyright for production use
